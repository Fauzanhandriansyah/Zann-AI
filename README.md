## ✨ Fitur Utama

Berdasarkan arsitektur yang dirancang, Zann AI memiliki 4 pilar fitur utama:

### 📸 1. Tutor Foto Tugas AI
Tidak perlu mengetik soal yang panjang. Cukup foto tugas Anda, dan biarkan AI menganalisisnya secara instan.
*   **Sumber Fleksibel:** Buka Kamera langsung atau Pilih dari Galeri.
*   **Kategori Spesifik:** Dukungan untuk Soal Matematika, Kode Program, dan Dokumen teks.
*   **Tindakan AI Cerdas:** Pilih apakah Anda ingin AI *Menjelaskan isi soal*, *Memberikan langkah penyelesaian*, atau *Menunjukkan letak kesalahan*.

### 💬 2. AI Chat & Pembelajaran Langsung (Live)
Interaksi yang mulus dan natural layaknya berbicara dengan tutor pribadi.
*   **Zann AI Live:** Fitur obrolan suara dua arah (*real-time voice interaction*).
*   **Riwayat Obrolan Cerdas:** Semua percakapan disimpan dengan rapi untuk diakses kembali kapan saja.
*   **Upload Konteks:** Upload file sebagai referensi untuk percakapan dengan AI.

### 📚 3. Alat Belajar Pribadi
Pantau dan tingkatkan progres belajar Anda dengan terstruktur.
*   **Rencana Belajar AI:** Buat *Study Plan* otomatis (contoh: Pengenalan Python) lengkap dengan persentase progres dan *checklist* subtopik.
*   **Sesi Flashcard:** Uji pemahaman Anda dengan *Flashcard Session*. Lacak skor pemahaman, kartu yang belum dikuasai, dan *generate* kartu baru secara otomatis.

### 🎨 4. Fitur Kreatif & Kustomisasi
Sesuaikan aplikasi dengan gaya dan kebutuhan Anda.
*   **Zann AI Memory:** Latih AI untuk mengingat basis pengetahuan spesifik Anda agar responsnya lebih personal dan akurat.
*   **Palet Tema Warna:** Bosan dengan warna bawaan? Ganti tema UI Anda (contoh: *Cyber Fusion*, *Sunset Horizon*, *Forest Moss*, *Cosmic Nebula*).

*   Node.js (v16+)

*   Kunci API (API Key) dari penyedia AI.

### Instalasi

1. **Clone Repositori ini:**
   ```bash
   git clone [https://github.com/Fauzanhandriansyah/zann-ai.git](https://github.com/Fauzanhandriansyah/zann-ai.git)
## Run Locally

**Prerequisites:**  [Android Studio](https://developer.android.com/studio)


1. Open Android Studio
2. Select **Open** and choose the directory containing this project
3. Allow Android Studio to fix any incompatibilities as it imports the project.
4. Create a file named `.env` in the project directory and set `GEMINI_API_KEY` in that file to your Gemini API key (see `.env.example` for an example)
5. Remove this line from the app's `build.gradle.kts` file: `signingConfig = signingConfigs.getByName("debugConfig")`
6. Run the app on an emulator or physical device
